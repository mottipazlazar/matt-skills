## Initial Campaign Templates

### NewLandWS_Template_1
- **Style:** Single-line, direct contact/record verification
- **Best for:** Standard cold outreach batches
- **Tone:** Professional, neutral, framed as data verification
- 4 messages + Alt Message with spinners

### NewLandWS_Template_2
- **Style:** Multi-line, softer confirmation format
- **Best for:** Alternate batches, carrier variation
- **Tone:** Slightly more conversational, same verification framing
- 4 messages + Alt Message with spinners

### AI_Direct_Purchase
- **Style:** Direct purchase interest, warmer tone
- **Best for:** Warmer lists or direct outreach
- **Tone:** More upfront about land acquisition interest
- 4 messages + Alt Message with spinners

## Followup Templates

### Landbuyer Follow Up 1
- **Use at:** D1 or D4 text steps
- **Tone:** Re-engagement, warmer, focused on scheduling a call
- **Best for:** Leads that have shown some prior engagement
- 4 messages + Alt Message

### Landbuyer Follow Up 2
- **Use at:** D1 or D4 text steps (alternate from Follow Up 1)
- **Tone:** Softer, curiosity-based, checking if they saw previous message
- **Best for:** Colder leads in followup, or when Follow Up 1 has already been used
- 4 messages + Alt Message

**LC 112hr Rule:** Always vary which message number and which template you use within any 112-hour window to avoid carrier filtering.
