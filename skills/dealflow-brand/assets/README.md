## Logos (Google Drive)
**Folder ID:** `https://drive.google.com/drive/folders/1--IWj1lvI8kZSQ1hu5CCep00vbW8qChR`

### Expected Files in assets/logos/
```
logo-horizontal-color.png     ← PRIMARY — use most often
logo-horizontal-white.png     ← On dark/photo backgrounds
logo-horizontal-grey.png      ← B&W print
logo-vertical-color.png       ← Square formats, profile
logo-vertical-white.png       ← Vertical on dark BG
logo-icon-color.png           ← Watermark, favicon, small
logo-icon-white.png           ← Small on dark
```

### Which logo version to use
| Background | Use |
|---|---|
| Light / white / Soft Ivory | logo-horizontal-color.png |
| Dark (Dusty Pine / Taupe) | logo-horizontal-white.png |
| Photo / busy image | logo-horizontal-white.png |
| B&W document | logo-horizontal-grey.png |
| Square social / stamp | logo-vertical-color.png |
| Watermark / favicon | logo-icon-color.png |

### Minimum Sizes (HARD RULES)
- Digital: **15px height minimum**
- Print: **6mm height minimum**
- Clear exclusion zone: icon height on all 4 sides
